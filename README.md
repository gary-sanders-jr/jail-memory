# jail-memory
For miss.fish
煎鱼的大门是汉白玉的大石碑。每次我们穿着宽大的、散发着劣质化纤味道的校服走进去时，在身后发出刺耳的人脸核验机的声音，就像是屠宰场里牲畜临死前的嘶吼。

我叫小鱼，在这里待了三年，第一年的，年级主任的名字还不是那个后来令人闻风丧胆的存在，小鱼很舒服的过了一年。

第二年，我变成了一个异类。我是年级主任和班主任最恨的那种人，因为我看他们的眼神里没有恐惧；但我又是他们最离不开的人，因为每一次月考、期中考、期末考，乃至最后的高考，我的名字永远用粗黑的记号笔写在班级红榜的最顶端，高高地悬挂在连接楼道与食堂的走廊里的”尖茅草“表彰墙上，像一面供他们炫耀的免死金牌。

高一刚进去的时候，学校就立下了规矩。在这里，时间不是用来感受的，是用来切割的。早上五点半，宿舍楼的电铃会像催命一样响起，那种尖锐的声音能直接刺破耳膜，扎进脑子里。你需要在一分钟内穿好衣服，三分钟内洗漱完毕，然后像被狗撵着的羊群一样冲向操场。

校长经常看不见，每次看脸上都挂着皮笑肉不笑的表情，据说经常被人看到和儿子一起往一些家属楼里乱窜；副校长高瘦，据说还是个诗人。他每周一早上就站在操场的主席台上，手里拿着一个高音喇叭。他最喜欢说的话是：“你们知道寒号鸟的故事吗？”哦，忘记说我们的主人公，年级主任jb了。

他的口水在清晨的冷空气里喷散。底下是上千个十四五岁的少年，每个人都低着头，手里捧着语文书或英语书，扯着嗓子嘶吼。jb会背着手，在班级的窗户外走来走去。如果谁的声音小了，或者谁的眼睛没有死死盯在书本上，他的手机就会狠狠地锁住那个人的头，发出一声清脆的一声“咔”。

我从高二就开始反抗。我不吼，我只是张着嘴，不出声。jb走到我面前，用戳我的胸口，恶狠狠地瞪着我：“哑巴了？”

我看着他浑浊的眼睛，平静地说：“嗓子疼。”

他举起手机想拍，但他停住了。因为前一天的摸底考试成绩刚出来，我是我们班第一。他的脸抽搐了一下，手机在半空中停了两秒，最后拍了我的同桌。那个好兄弟明明在很用力的背书。

jb指着那个无辜的男生骂道：“看什么看！你如果有他（指我）那个成绩，你也可以当哑巴！”，并在年级大群里通报了这个男生的”所作所为“。

从那天起我就明白了这座学校的底层逻辑：这里没有尊严，没有公平，没有教育，只有赤裸裸的交易。分数就是硬通货，就是武器。只要你的分数足够高，你就可以在一定程度上豁免这台绞肉机的物理伤害。

但我看到了更多的人，那些没有分数护体的人，是如何被一层层剥掉人皮的。

分班之后，我们班换了一个数学老师。他名叫金山，是个五十来岁的中年人，腰背总是习惯性地佝偻着，像是被什么看不见的东西压着。他总是穿着一件洗得发黄的白衬衫，袖口磨得开了线。金山其实是个和蔼的老头，讲课时声音不大，慢条斯理的，但毕竟上了年纪，讲到复杂的解析几何时，额头上总会沁出一层细密的汗珠。

那种和蔼，在年级主任jb面前就像一张薄得透明的纸。

我的同桌叫王驴。他是我见过最努力的人。他每天午休从不睡觉，别人去打球、去小卖部，他就在那座用书堆成的坟墓里坐着。为了省时间，他连午饭都请人带，一到晚上，他的课桌周围就飘着一种三明治的香味，还有那种廉价圆珠笔用完后残留的塑胶味。他的草稿纸总是写得密密麻麻，像是一群黑色的蚂蚁在纸上拼命挣扎。

那天早上，金山正站在讲台上对刚考完的月考的答案。王强低着头，正对着卷子上那三道空白的大题发愣。

教室后门那扇一直关不严的门，突然“砰”地一声被撞开了。

年级主任走了进来。他那天穿了一双黑色的皮鞋，在刚拖过的地上踏出的声音格外清脆，像是一声声敲在人的脊梁骨上。金山的声音戛然而止，他下意识地站直了身子，手里的半截粉笔掉在地上，碎成了几瓣。

年级主任没理会老师，他径直走到王驴身后。他并没有立刻说话，只是伸出那只肥厚的手，猛地抓起王驴的卷子，扯到自己眼前。

“三道大题，一笔没动？”年级主任的声音在死寂的教室里回荡，带着一种猫戏弄耗子般的戏谑。

王驴吓得猛地站起来，凳子在地上划出一声刺耳的尖叫。他的双手死死地抓着裤缝，指关节因为用力过度而呈现出一种惨白色。

“金老师，这种学生你还留着干什么？”年级主任转过头，冷冷地瞥了金山一眼。金山张了张嘴，脸上的皱纹局促地抖动着，最后什么也没说，只是低头去捡那截断掉的粉笔。

年级主任把卷子扔到桌子上。

“你每天是在这里认真学吗？”年级主任猛地跨前一步，肚子顶到了课桌边缘，“每天坐在位子上装模作样，我看你不是在做题，是脑子不知道飞哪去了。我昨天在监控里看你，你盯着窗外看了三秒钟，你在看什么？看女人？”

全班没有一个人敢抬头。我能感觉到王驴在发抖，那种抖动顺着并排的课桌传到我的手臂上，频率极快。他的脸涨成了猪肝色，眼泪在眼眶里打转，但他死死地咬着牙，不敢让它掉下来。

“哭？你还有脸哭？”年级主任发出一声短促的冷笑，他指着教室外那条阴冷的走廊，“走廊外站着去，等你护犊子的班主任回来捞你。”

年级主任说完，又环视了一圈教室，那目光像是一把带钩子的秤，在每个人的脖子上晃了一圈，才施施然地走了出去。

金山叹了口气，他看着王驴，眼神里闪过一丝不忍，但随即又变成了那种习惯性的木然。他摆摆手，示意王强出去。

王驴像一条被抽去了脊椎的狗，低着头，慢慢地蹭出了教室。

那天中午，走廊里只有穿堂风。我带了一份饭去给王强，他坐在走廊冰冷的瓷砖地上，手里攥着笔，由于长时间的罚站和饥饿，他的手抖得比早上还要厉害。他在纸上写下的每个字都歪歪扭扭，纸面上全是眼泪干透后留下的褶皱。

我问他：“他凭什么这么侮辱你？”

王驴没有抬头，他的声音很轻，像是被风一吹就会散掉：“我成绩差，我活该。”

我看着他，又看了看教室门楣上贴着的“人生在琴”四字。在那一刻我意识到，jb不需要我们成为人才，也许只需要我们听话。他通过羞辱王驴，完成了对我们所有人的驯化。他让我们相信，分数的高低决定了人格的高低。如果你考得不好，你就不仅是学业失败，你整个人就是一种罪恶。

而那些像金山和我们班主任学潮一样和蔼却软弱的老师，他们只能在这种高压下保持沉默，眼睁睁地看着自己的学生被这种权力碾碎。

哦对了，讲个题外话，我们的班主任叫学潮，物理老师，讲课非常好，或者说我感到幸运，我们班每个老师讲课都非常好，他和jb不对付，原因是他的爱人举报jb在外面补课，因此jb记恨于心。

那一刻，我感到一阵深深的恶寒。学校和家长联合起来，用一种名为“未来”的幻觉，把这群年轻人的骨头一点点敲碎。他们让王驴觉得，自己受到的所有屈辱都是理所应当的，是因为自己“不够好”。这是一种极其狠毒的洗脑。

我开始用我自己的方式报复这个系统。

我不听课，上课的时候我用书本挡着，在下面看文学作品、网络小说，毕业的时候我的橱子里全是书，光余华的就有四五本。我不写老师布置的那些重复性、机械性的作业。每次收作业，我都说我没写。

学潮把我叫到办公室：“你以为你聪明就了不起吗？你这种态度，迟早要摔大跟头！你为什么不写作业？”

我站在他的办公桌前，看着他愤怒到扭曲的脸，心里没有任何波澜。我说：“老师我只是忘记写了。”

他气得浑身发抖，指着我的鼻子骂我狂妄、自私、没有集体荣誉感。他把我晾在办公室里站了整整两节课，期间每一个进小办公室的老师和同学都要对我冷嘲热讽几句。他们试图用集体的口水把我淹没，试图在我脸上看到羞愧和屈辱。

但我没有。我只是冷冷地看着他们。我知道他们为什么愤怒。他们愤怒的不是我不写作业，而是我脱离了他们的掌控。在这个学校里，所有的规矩——罚站、抄写、跑操、宣誓——本质上都不是为了让你学到知识，而是为了“规训”。他们需要看到你低头，看到你恐惧，看到你为了哪怕一分而在他们面前卑躬屈膝。

而我，剥夺了他们享受这种权力的快感。

高二最后一场济南市联考，成绩出来，我依然是班级第一。学潮在班会上用他蹩脚的表达，想了半天憋出来一句，“有些同学很不好啊”。但是jb刚刚在大会上表扬了他，说他带出了一个不错的苗子。

这就是现实的荒谬。他们一边恨不得把我踩在脚下，一边又不得不把我供起来。我成了一个他们无法消化、又不敢吐出来的铁核桃。

到了高三，整栋教学楼的气味都变了。那是一种混合着发酵的汗酸味、劣质咖啡味、甚至还有隐隐约约的风油精味儿。

所有人都在发疯的边缘。

走廊里贴满了红色的横幅：“提高一分，干掉千人”，“只要学不死，就往死里学”。这些标语像是一张张血盆大口，贴在墙上，每天吞噬着我们的生气。

女生的头发大把大把地掉，男生开始长满脸的痘。我前面的那个女生，每天晚上在宿舍里拿指甲抠自己的胳膊，为了让自己在凌晨能在被窝里学习的时候不睡着。有一天夏天，她穿了一件短袖，我看到了她手臂上密密麻麻的、像紫红色蜈蚣一样的疤痕。

但最惨烈的，是那种无声的崩溃。

不知从什么时候开始，好像是一轮复习，年级在jb的带领下，开始了一场荒诞的封建帝制建设。在他治下，原本的春夏、韩韬、标标三位副主任显然已经不够用来镇压庞大的高三群体了，底层管理人员像繁衍的真菌一样迅速膨胀，底下又多出了董卓、阿震等一批副副主任。

这群人每天端着不锈钢保温杯，皮鞋在地板上踏出沉闷的回音，像一群巡视领地的王侯。他们把整个高三楼划分成了各自的藩镇，监控摄像头和"督查组"是他们不知疲倦的眼睛，大喇叭的通报是他们的圣旨。

因为我成绩较好却又总是不服管教，我成了这套帝国官僚系统里最难处理的钉子。他们不能开除我，因为红榜最顶端需要我的名字来装点门面；但他们又必须羞辱我，以维持他们摇摇欲坠的威严。于是，围绕着我，他们发明了一系列常人无法理解的刑罚。

煎鱼的晚修是不能交流的，我在熄灯哨后，小腿不自觉的抽筋，做起来掰脚。

jb正好从后门的玻璃窗外走过。在这位土皇帝的眼里，他看不到学生的抽筋，他只看到了一个没有保持绝对睡姿的异类。他猛地推开门，夹带着走廊里的冷风走了进来。

“你在干什么？”他居高临下地看着我。
“抽筋。”我如实回答。

jb冷笑了一声，”还找借口？那就回家治疗抽筋，什么时候治好了再回来上课！！“

然后他弯腰，用他那张令人恐惧的脸贴近我，”现在你还抽筋吗？我来给你治治吧。“

jb治疗抽筋的办法就是让我在楼道里站着，冬天的楼道开着窗户，冷的令人腿上没有直觉。我看到jb站在避风的角落里，一边搓手一边盯着我，他的眼神里有一种执行私刑的快感。

他们以为这样就能让我感到被抛弃的恐惧。但我只觉得安静。

有一天下午的自习课，我做完了所有的卷子，觉得无聊极了。我转过身，面对着那面剥落了一块白灰的后墙。墙上有一道裂缝，形状像是一条干涸的河床。我盯着那条裂缝，也就盯了几秒。

天花板上的监控探头转动了一下，红灯闪烁。

十分钟后，董卓气势汹汹地冲进教室，一把揪住我的后衣领，把我拖进了年级主任办公室。

“上课不看书，坐在最后一排回头说话！”董卓拍着桌子，口沫横飞，仿佛抓住了我叛国的证据。

我看着他，觉得极其滑稽，平静地说：“我坐在最后一排，后面只有墙。我能和谁说话？和鬼吗？”

jb坐在宽大的办公桌后，吹了吹杯子里的枸杞，头也不抬地说：“和主任顶嘴，这是对学校纪律的公然蔑视，是在精神上反抗年级。”

于是，我又被罚站了一下午。

但这些处罚加起来，都不如那张贴在年级布告栏里的通报批评来得荒谬。

那是在离高考还有两白天的时候，我在教室里转笔。

韩韬当时正像幽灵一样在走廊里巡视。他冲进来，如获至宝地缴获了我的圆珠笔。

他们知道我成绩好，普通的违纪通报已经对我没有任何震慑力，他们需要用最耸人听闻的罪名来把我钉在耻辱柱上，借此向两千多名高三学生展示帝国权力的不可违抗。

第二天，一张盖着年级鲜红大印的通报贴在了级部办公室门口最显眼的地方。上面用黑体大字写着：

“高三29班小鱼同学，无视年级纪律，上课期间不思进取，在座位上公然玩耍原子弹。鉴于其行为已严重威胁到周围同学的生命安全与学习氛围，给予严重警告处分一次。”

不知道多少同学从级部办公室门口走过，都仰着头看那张通报。令人毛骨悚然的是，没有人笑。在这个高压的、荒诞的系统里，“玩原子弹”这个罪名竟然被大家默默地接受了。他们已经被规训得失去了常识和质疑的能力，只要是盖着年级公章的白纸黑字，哪怕说我在教室里发射了航天飞机，他们也会深信不疑。

我站在人群里，看着那三个字——“原子弹”。我没有觉得屈辱，我只觉得这群管理者可悲到了极点。他们就像是一群穿着龙袍的猴子，用尽了所有荒诞不经的词汇，只为了掩饰他们对一个不服从的年轻灵魂的恐惧。

那张通报贴了整整一个月，没有被撕下来，也没有被其他的通报压在下面。

六月的风终于吹进那扇大门，那场被他们渲染成生杀大权的考试，其实结束得异常沉闷。没有电影里那种撕碎试卷、仰天长啸的狂欢，整栋高三楼静得像一座刚刚被洗劫过的废墟。大家只是麻木地收拾着东西，太累了，连多说一句话的力气都没有。

我背着轻飘飘的书包走出校门，远远地看见jb、董卓和阿震他们站在台阶上。失去了“高三”这个庞大的宿主，这群曾经不可一世的帝国官僚，突然缩成了几个干瘪、疲惫且毫无光泽的中年男人。他们手里的麦克风哑了，那种掌控别人命运的绝对权力，在那天下午，像断了电的绞肉机一样轰然停转。

我连头都没回。我知道，红榜上的名字很快会被撕下来，这座铁笼很快又会迎来新一批的祭品。

后来听说，我们年级考的非常好，学校论功行赏，韩韬等副主任，晋升为主任，董卓等副副主任（一想到这个词我就想笑），晋升为副主任；至于和善到无法融入的标标副主任，则被jb发配”晋升“，去采食做主任去喽。

这就是我为什么要写下这篇长文。
